To set up a new project with git/github/npm/jest run the following commands in the terminal:
    npm install
    npm init
    jest --init
    git init
    git add .
    git commit -m 'commit message'
    git branch -M main
    git remote add origin "git@github.com:bjhufstetler/observatory.git"
    git push -u origin main

To use tle.js 
    npm i tle.js@3
